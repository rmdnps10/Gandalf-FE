## Overview

자연어 입력을 기반으로 대학 커뮤니티 내 인재를 검색하는 AI 플랫폼.
검색 → 프로필 → 연락 흐름을 하나의 UX로 연결.

---

## Features

* 자연어 기반 인재 검색
* AI 의도 분석 기반 결과 추천
* 프로필 관리
* JWT 인증 시스템
* Google OAuth 로그인
* 이메일 연락 연결
